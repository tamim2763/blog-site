### Simple Blog Project

A simple blog application built with Node.js and Express. This was a learning project to understand the fundamentals of backend web development, routing, and CRUD operations.

Blogs are stored temporarily in server memory and will be reset if the server restarts.

### Features

* **Create:** Write and publish new blog posts.
* **Read:** View all published posts (with a "Read More" snippet) and view a full post on its own page.
* **Update:** Edit existing posts.
* **Delete:** Remove posts.

### Tech Stack

* **Backend:** Node.js, Express.js
* **Templating:** EJS (Embedded JavaScript)
* **Middleware:** body-parser
* **Styling:** HTML, CSS, Bootstrap
* **Client-side JS:** jQuery (for random-colored backgrounds)

### How to Run This Project Locally

1.  Clone the repository:
    ```bash
    git clone [https://github.com/tamim2763/blog-site.git](https://github.com/tamim2763/blog-site.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd blog-site
    ```
3.  Install the necessary dependencies:
    ```bash
    npm install
    ```
4.  Run the server:
    ```bash
    node index.js
    ```
5.  Open your browser and go to `http://localhost:3000`.
